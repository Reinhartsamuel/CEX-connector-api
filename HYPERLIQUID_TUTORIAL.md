

How to crete and approve Agent Wallet on Hyperliquid
1. Create a new wallet
2. go to https://app.hyperliquid.xyz/API
3. give name and address of that newly created wallet
4. click "Approve API Wallet"
5. set MAX days valid
6. Authorize
7. Sign transaction on wallet
8. Copy master wallet addres to byscript as api_key
9. Copy agent wallet private key as api_secret
